花界
==============================

介绍
--------
- 收集各种花的图片
- 能组合拼图，与朋友分享
- 整理花语，养花技巧，花朵的百科知识，散文诗词等等
- 提供下载收藏
- 它是一个免费开源的Android移动应用，方便快捷
